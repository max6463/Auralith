# Auralith

Auralith is a lightweight platform for real-time data acquisition and log intelligence, designed to turn raw system signals into clear, actionable insights.

## Backend

The backend is built with **CodeIgniter 3 (PHP)** and uses a migration-based database architecture.

It provides:
- API key authentication
- Structured log ingestion
- Minimal, fast API layer

## Focus

Built for engineers working with telemetry, event streams, and data acquisition systems.
